# Simple Image Effector

Simple image effector made with Vanilla Javascript.  
[[Demo]](https://coachoox.github.io/js-image-effector/)  

## Preview

![](./images/preview.png)

## Features

-   Upload Image
-   Apply Filters
    -   Brightness
    -   Contrast
    -   Blur
    -   Gray Scale
    -   Hue Rotate
    -   Invert
    -   Sepia
